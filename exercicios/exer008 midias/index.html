Imagens dinâmicas

Tente abrir esse site em vários dispositivos diferentes ou simplesmente mude o tamanho da tela

A tag picture permite que você crie vários sources para definir qual imagem vai aparecer

Dentro do source media type, se o tamanho for no maximo 1000 eu vou carregar a m, se for maior que 1000 ele vai carregar a g de baixo, ele vai começar com min-width aí vc muda para max, e digita o caminho da imagem em srcset, e no type coloca o tipo da imagem, no caso png sempre coloca os sources acima da imagem maior

Reproduzindo aúdio

Ele primeiro vai tentar carregar o primeiro arquivo no primeiro formato no caso o mp3, se não conseguir ele vai indo para o de baixo se não conseguir nenhum ele vai exibir a mensagem

Na tag audio, dentro do preload se colocar auto, o site vai esperar carregar tudo pra tocar, se colocar metadata ele vai carregar as informações necessárias e quando o usuário colocar para tocar que vai carregar o aúdio, se colocar none o site não vai carregar nada, o loop é para ele ficar tocando toda vez que acabar
