Estilos css externos

Para fazer o css externo é só digitar link em head e escolher link css, ao passar o mouse por cima segura ctrl e clica no arquivo que é para gerar, e para copiar a formatação para outros sites é só copiar e colar o link no cabeçalho

posso fazer os 3 estilos se quiser, o de maior importância para o html é o inline pois é o que vai se sobressair sobre os outros, logo depois o interno, e por último o externo, mas em termos de usabilidade é melhor seguir a ordem contrária, a não ser que tenha algumas formatações pontuais que vão diferir do padrão.
